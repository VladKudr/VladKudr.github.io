OpenClaw + Telegram

Создаём Telegram-бота с нуля и подключаем его к OpenClaw. Никаких специальных знаний не нужно — только Telegram и 10 минут.

~10 минут
Telegram + VPS
Для начинающих
Содержание
  1. Что такое бот и зачем это нужно
  2. Создаём бота через @BotFather
  3. Добавляем токен в OpenClaw
  4. Сопрягаем бота с собой (pairing)
  5. Проверяем — пишем первое сообщение
  6. Добавить бота в групповой чат
1

Что такое бот и зачем это нужно

В Telegram можно создать собственного бота — это аккаунт-автоответчик, которым управляет программа на сервере. Ты создаёшь «оболочку» бота в Telegram (это займёт 2 минуты), а OpenClaw становится его мозгом.

После настройки ты сможешь писать своему агенту прямо из Telegram — с телефона, планшета, любого устройства. Сервер работает круглосуточно, агент всегда онлайн.

ℹ️
Перед началом

Этот гайд продолжает «Самостоятельную установку OpenClaw». Убедись, что OpenClaw уже установлен на сервере и Gateway запущен.

2

Создаём бота через @BotFather

@BotFather — это официальный сервис Telegram для создания ботов. Всё делается прямо в мессенджере, ничего скачивать не нужно.

1

Найди @BotFather в Telegram

Открой Telegram, в строке поиска введи @BotFather. Убедись, что рядом с именем стоит синяя галочка — это официальный аккаунт Telegram, а не копия.

2

Отправь команду /newbot

Напиши в чат с BotFather:

/newbot

🤖

21:34
Мой ассистент
21:35

21:35
Написать...
3

Придумай имя и username бота

Имя — как будет называться бот в чатах. Например: Мой ассистент. Можно по-русски, любое.

Username — уникальный адрес, только латиницей, обязательно заканчивается на bot. Например: my_assistant_vlad_bot

4

Сохрани токен — это главное

После создания BotFather пришлёт токен — длинную строку. Это единственный «ключ» от твоего бота.

Токен выглядит так: 1234567890:AAFzQ8k3NjhPm.... Сохрани его в заметки — он понадобится на следующем шаге.

3

Добавляем токен в OpenClaw

Теперь переходим на сервер. Подключись по SSH так же, как делал при установке.

1

Открой файл настроек OpenClaw

terminal — ssh на сервер 
nano ~/.openclaw/openclaw.json
2

Добавь блок для Telegram

Найди в файле закрывающую скобку } и добавь секцию channels. Итоговый файл должен выглядеть так:

~/.openclaw/openclaw.json 
{
  "model": {
    "provider": "moonshot",
    "name": "kimi-k2.5",
    "apiKey": "ТУТ_ТВОЙ_KIMI_КЛЮЧ"
  },
  "channels": {
    "telegram": {
      "enabled": true,
      "botToken": "ТУТ_ТОКЕН_ОТ_BOTFATHER",
      "dmPolicy": "pairing"
    }
  }
}

Замени ТУТ_ТОКЕН_ОТ_BOTFATHER на токен из шага 2. Остальное не трогай.

3

Сохрани файл и перезапусти Gateway

В редакторе nano: Ctrl+O → Enter → Ctrl+X. Затем:

terminal — ssh на сервер 
openclaw gateway restart
Терминал — сервер
root@vps-openclaw — ssh
root@vps:~# openclaw gateway restart Stopping gateway... Starting gateway... ✓ Gateway started (pid 4821) ✓ Telegram channel connected Listening on 127.0.0.1:18789   root@vps:~#
4

Сопрягаем бота с собой

OpenClaw использует систему сопряжения (pairing): первый раз нужно «представиться» боту, чтобы он понял, что ты — его хозяин, и отвечал только тебе.

1

Открой бота в Telegram и нажми Start

Найди своего бота в Telegram по username, который задавал в шаге 2 (например @my_assistant_vlad_bot). Открой диалог и нажми кнопку Start или отправь /start.

2

Получи код сопряжения

Бот пришлёт твой Telegram ID и код сопряжения — несколько букв и цифр.

Telegram — первое сообщение от бота
🧠
Мой ассистент
бот
/start
21:38
👋 OpenClaw is running!

Your Telegram ID: 381927465

Pairing code:
X4K-892

Run on your server:
openclaw pairing approve telegram X4K-892
21:38
Написать...
3

Подтверди на сервере

Вернись в терминал (SSH на сервер) и введи команду с кодом из Telegram:

terminal — ssh на сервер 
openclaw pairing approve telegram X4K-892

Замени X4K-892 на реальный код из сообщения бота.

Терминал — подтверждение сопряжения
root@vps-openclaw — ssh
root@vps:~# openclaw pairing approve telegram X4K-892 Looking up pairing request... User: 381927465 ✓ Pairing approved. User added to allowlist.   root@vps:~#
⚠️
Код живёт 1 час

Если не успел подтвердить — просто отправь боту /start ещё раз, он пришлёт новый код.

5

Проверяем — пишем первое сообщение

Всё готово. Напиши боту любое сообщение в Telegram и агент ответит через несколько секунд.

🧠

Если агент не отвечает — проверь логи на сервере: openclaw logs --follow. Там увидишь причину в реальном времени.

6

Добавить бота в групповой чат

Хочешь, чтобы бот работал в общем чате с коллегами или друзьями? Три простых действия.

1

Добавь бота в группу

Открой групповой чат в Telegram → Участники → Добавить участника → найди своего бота по username.

2

Разреши боту читать все сообщения

По умолчанию Telegram разрешает боту видеть только сообщения, начинающиеся с /. Чтобы он отвечал на любые обращения — открой диалог с @BotFather и отправь:

/setprivacy

Выбери своего бота → выбери Disable. Это разрешит боту видеть все сообщения в группе.

3

Переподключи бота к группе

После изменения настроек приватности нужно удалить бота из группы и добавить снова — только тогда Telegram применит новые права.

💡
В группе бот отвечает по упоминанию

По умолчанию в групповом чате бот реагирует только когда его упоминают: @my_assistant_vlad_bot расскажи про.... Это удобно — он не мешает обычным разговорам.
